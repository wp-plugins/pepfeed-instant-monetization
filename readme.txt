The PepFeed Instant Monetization WP Plugin automatically detects the gadgets on your reviews and displays all the deals available for each product.

== Description ==

The PepFeed Instant Monetization WP Plugin automatically detects the product names mentioned on your page’s titles and headers to display the top deals for each gadget. Once installed, no additional setup is required. The plugin matches the product names on each page and it takes effect across all the pages in your website. You can enhance the user experience of your website’s visitors by showing them the best deals they would usually search for on a different website, while keeping the commission entirely for yourself.

= Key Features =
* Show deals from world's leading retailer, Amazon.com
* Include your affiliate tag, you keep 100% of the commissions!
* Deals updated regularly and automatically
* Our database recognizes more than 10,000 different electronic products, and grows everyday
* Instantly improve your website with the best deals to complement your reviews


== Installation ==

1. Install PepFeed either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Get your Amazon affiliate ID at [http://affiliate-program.amazon.com/gp/associates/apply/main.html](http://affiliate-program.amazon.com/gp/associates/apply/main.html)
3. After activating PepFeed, go to the Settings page to enable all the PepFeed features and input your affiliate ID.
4. Click the checkbox to accept Terms and Conditions and get full access to all PepFeed's features
5. That's it! Now go write great tech reviews and make more money!


== Frequently Asked Questions ==

= How many deals can users see? =
* You can see as many deals as available for that product.

= What kind of products does it search deals for? =
* Consumer Electronics ranging from smartphones to tablets and computers, as well as small household electronics: electric kettles, all sorts of food blenders and mixers, microwaves, etc...

= Which browsers is it compatible with? =
* The PepFeed WP Plugin works perfectly regardless of the browser used to open your website and it’s responsive for a 150px width (as this is still a Beta version).

= Which countries/regions does it get deals for? =
* At the moment, it is getting the best deals for the United States.
